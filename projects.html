Bj's Party House Intake Form

The goal of this project was a complete redesign of the client intake for Bj's partyhouse, a locale clown, and a performer troupe. During the length of this project, I designed and implemented the intake form currently on their page.

Before this project, performer requests and contract creation were done entirely manually. Through this process, I modified the Wix page to make a comprehensive and flexible form where the output is an email that serves as a template contract.

Some of the challenges that I faced in this project were my unfamiliarity with the Wix platform, the fact that the platform had been designed by previous students and had some major bugs, and my clients being unsure of what exactly they wanted.

In the end, there are still some things that I could have improved on, like the readability of the form, but overall, it worked and the clients liked it, and that was what mattered.

One feature I am most impressed with is the price estimation calculator I built into the system.

Self Hosted PhP & MySql web Aplcation

This project was started when I was having some trouble navigating the wiki for a lion collection game I play. As I was looking through the wiki, I thought this was an amazing dataset, and having some web application that could list the requirements for a base just by entering its name would be nice.

Thus, I started brushing up on my PHP and leveraging my SQL skills and ended up. with this application. The whole thing is hosted on a Pi4 and is run with MaranaDB and PhPMyadmin. The function takes an input, compares the name to the database of names, then replies with what kind of base was entered and returns other important infromation.

This project taught me a lot about connecting PHP and SQL, designing SQL statements for use in web-based applications, and hosting a web server. There is more I plan to do with this project, such as expanding the search function, and posably alowing for more complex serches.
